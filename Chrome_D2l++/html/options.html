Note

This extension was inspired by Better Canvas and Darklight. Since both do not provide dark themes and other features to the D2L learning platform, I decided to create this extension. Some parts of the platform may not apply the necessary themes. This could be due to not having access to specific selectors or being unable to find them. I will add these in future updates if possible. If you would like to contribute and improve the extension, there are steps you can follow.
